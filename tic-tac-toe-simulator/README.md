# Tic-Tac-Toe Simulator

A C# Windows Forms application that simulates a tic-tac-toe game by filling the board automatically, tracking board state, and detecting winning combinations.

This project highlights logic design in a desktop UI by combining randomization, a 2D array for board storage, and row, column, and diagonal winner checks.

## Features

- Starts a new randomized tic-tac-toe game
- Tracks the board using a 2D character array
- Detects wins across rows, columns, and diagonals
- Highlights the winning path in the interface
- Displays tie-game results when no winner is found

## Built With

- C#
- Windows Forms
- .NET Framework 4.7.2

## Concepts Demonstrated

- Two-dimensional arrays
- Control flow and game-state logic
- Randomized board population
- Win-condition checking
- Windows Forms event handling

## How to Run

1. Open the project in Visual Studio on Windows.
2. Load `Project7.csproj`.
3. Build and run the application.
