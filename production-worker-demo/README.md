# Production Worker Class Demo

A C# Windows Forms project that collects employee information and demonstrates inheritance through an `Employee` base class and a `ProductionWorker` derived class.

This project was created to practice object-oriented programming concepts in C#, especially encapsulation, inheritance, and using form input to populate class properties.

## Features

- Accepts employee name, number, shift, and hourly rate
- Stores values in custom C# classes
- Demonstrates inheritance with `ProductionWorker : Employee`
- Displays formatted employee information in the UI

## Built With

- C#
- Windows Forms
- .NET Framework 4.7.2

## Concepts Demonstrated

- Classes and objects
- Inheritance
- Properties and encapsulation
- User input parsing
- Windows Forms event-driven programming

## How to Run

1. Open the project in Visual Studio on Windows.
2. Load `Project10.csproj`.
3. Build and run the application.
