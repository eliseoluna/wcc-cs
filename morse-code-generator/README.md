# Morse Code Generator

A C# Windows Forms application that converts plain text into International Morse code.

This project was built as part of a C# coursework sequence and demonstrates event-driven desktop development, input validation, and character-by-character text conversion.

## Features

- Converts letters, numbers, and selected punctuation into Morse code
- Validates keyboard input before conversion
- Displays formatted Morse output in the app interface
- Supports clearing the current input and output

## Built With
- C#
- Windows Forms
- .NET Framework 4.7.2

## Concepts Demonstrated

- Event-driven programming
- Form controls and UI state management
- Regular expression input validation
- Array-based character lookup
- String formatting and text processing

## How to Run

1. Open the project in Visual Studio on Windows.
2. Load `WindowsFormsApp1.csproj`.
3. Build and run the application.

## Notes

- This is a desktop Windows Forms project targeting .NET Framework 4.7.2.
- The application converts text to Morse code output and is intended as a learning project focused on core C# and GUI development.
